# 店舗作業タイマー

冷凍食品の店舗配送・陳列ワークの作業時間を、ストップウォッチ式で計測する個人用 PWA。
「どこで止まっているか」を per-unit（○○あたりの速さ）で発見するのが目的。

- 公開URL: https://sebanne1225.github.io/store-timer/
- データは端末の `localStorage` に保存（サーバー送信なし）
- JSON でエクスポート / インポート（バックアップ・復元）

## 使い方
1. 日付・今日の EOS を入力
2. 店を選び、数量（任意）を入力
3. ▶開始 →「次へ」でステップをシームレスに計測
4. 一時停止 / ◀戻る / スキップ / ステップ手修正 / 🗑店ごとリセット
5. 店完了で per-unit（○○あたり, 分:秒）が出る

計測対象は「店到着〜出発の作業時間」（車の移動時間は測らない・異常はメモ欄へ）。

## per-unit の式
- ケースあたり = (アイス+冷食 時間) / (アイス数+冷食数)
- アイスあたり = アイス時間 / アイス数 ／ 冷食あたり = 冷食時間 / 冷食数
- バラあたり = バラ時間 / バラ数 ／ オリコンあたり = バラ時間 / オリコン数
- S店 = 合計時間 / ケース数

## 店プリセット
- A: 冷食 / アイス
- C: 積み下ろし+検品 / アイス / バラ / 冷食 / 前出し / チェック+片付け / 週次(任意・合計外)
- S: 積み下ろし / 検品+陳列
- 現コース: A-K → C-T → S-T → S-F → C-A → C-N
